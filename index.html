דף הבית חיפוש לפי תמונה בניית מאגר
מוכן ל-GitHub Pages

חיפוש פנים באתר סטטי, בלי שרת.

בגרסה הזאת החיפוש מתבצע ישירות בדפדפן. אפשר להעלות תמונה, להשוות אותה למאגר שפרסמת, ולקבל התאמות מיידיות בלי שרת פייתון.

פתיחת חיפוש בניית המאגר שלך
GitHub Pages יכול לארח את האתר עצמו, אבל הוא לא מפעיל Backend של FastAPI ולא שומר העלאות ציבוריות משותפות. למאגר חי ומשותף עדיין תצטרכי אחסון אמיתי בהמשך.

מה שודרג כאן

• התאמות מיידיות עם כרטיסי תוצאות ברורים והדגשה להתאמה חזקה.
• דחיסת תמונות מקומית לפני עיבוד כדי להאיץ תמונות כבדות מהטלפון.
• Service Worker עם טעינת רשת תחילה ונפילה למטמון.

כלי שטח

• אפשרות לשמור מיקום נוכחי.
• כפתור הכנה מהיר לדיווח למוקד 106 דרך אפליקציית המייל.
• ממשק RTL מלא עם גופנים עבריים נוחים יותר.

מה לא היה בקוד הזה

• מפה עם Leaflet ונעיצת סמן ידנית.
• API של דיווחי "נמצא"/"אבד" וטלפונים.
• Backend להשוואות שרת או שמירת קבצים משותפת.
שלב 1

יצירת מאגר הפנים

פתחי את בניית מאגר, העלי תמונות דוגמה לכל אדם, ואז ייצאי קובץ JSON.
שלב 2

פרסום המאגר

העלי את הנתונים שייצאת אל data/library.json והוסיפי גם תמונות ממוזערות לריפו.
שלב 3

שיתוף עמוד החיפוש

שלחי אנשים אל /search.html כדי שיוכלו להעלות תמונה אחת ולקבל התאמות מדורגות.

דף הבית חיפוש לפי תמונה בניית מאגר
מוכן ל-GitHub Pages

חיפוש פנים באתר סטטי, בלי שרת.

בגרסה הזאת החיפוש מתבצע ישירות בדפדפן. אפשר להעלות תמונה, להשוות אותה למאגר שפרסמת, ולקבל התאמות בלי שרת פייתון.

GitHub Pages יכול לארח את האתר עצמו, אבל הוא לא מפעיל את ה-Backend של FastAPI ולא שומר העלאות ציבוריות משותפות. למאגר חי ומשותף עדיין תצטרכי אחסון אמיתי בהמשך.

מה עובד כבר עכשיו

• מבקרים יכולים להעלות תמונה אחת ולחפש במאגר שפרסמת ישירות מהדפדפן.
• אפשר לבנות רשומות בדיקה מקומית דרך עמוד בניית מאגר.
• אפשר לייצא JSON ולפרסם אותו בתוך data/library.json כדי שכולם ישתמשו בו.
• אין עלות Backend בשביל הדמו הסטטי.

מה עדיין יצריך אחסון אמיתי

• העלאות משותפות למאגר מרכזי אחד
• חשבונות ניהול פרטיים
• אחסון קבוע לתמונות שנשלחו
• עיבוד כבד ברקע לסרטונים או ייבוא בכמות גדולה
שלב 1

יצירת מאגר הפנים

פתחי את בניית מאגר, העלי תמונות דוגמה לכל אדם, ואז ייצאי קובץ JSON.
שלב 2

פרסום המאגר

העלי את הנתונים שייצאת אל data/library.json והוסיפי גם תמונות ממוזערות לריפו.
שלב 3

שיתוף עמוד החיפוש

שלחי אנשים אל /search.html כדי שיוכלו להעלות תמונה אחת ולקבל התאמות מדורגות.

מבנה קבצים

your-repo/
├─ index.html
├─ search.html
├─ enroll.html
├─ assets/
│  ├─ styles.css
│  └─ common.js
└─ data/
   └─ library.json

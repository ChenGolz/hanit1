בניית רשומות מאגר בתוך הדפדפן

העמוד הזה מיועד לך ולא למבקרים. מעלים תמונות דוגמה של אדם אחד, מחשבים תיאורי פנים בדפדפן, ואז מייצאים JSON עבור data/library.json.

יצירת רשומה אחת

שדות חובה: שם האדם ולפחות תמונה אחת.
הרשומות שנוצרות כאן נשמרות רק בדפדפן הזה עד שמייצאים אותן. כדי לפרסם אותן לכולם צריך לעשות commit לקובץ ה-JSON המיוצא בתוך הריפו.

מה חדש בתהליך

1. תמונות גדולות נדחסות מקומית לפני העיבוד כדי לחסוך זמן.
2. רשומה קיימת עם אותו שם מתמזגת במקום ליצור כפילות חדשה.
3. הממשק נשמר גם ללא רשת אם העמוד כבר נטען בעבר.
אחרי ייצוא הקובץ, החליפי את data/library.json בריפו ועשי commit כדי לפרסם את המאגר החדש.

יצירת רשומה אחת

שדות חובה: שם האדם ולפחות תמונה אחת.
הרשומות שנוצרות כאן נשמרות רק בדפדפן הזה עד שמייצאים אותן. כדי לפרסם אותן לכולם צריך לעשות commit לקובץ ה-JSON המיוצא בתוך הריפו.

איך לפרסם לכולם

1. ייצאי את קובץ ה-JSON מהעמוד הזה.
2. החליפי את data/library.json בריפו בקובץ שייצאת.
3. דחפי את הריפו ל-GitHub Pages.
4. שתפי את search.html.
לספרייה ציבורית מלוטשת יותר, כדאי להוסיף גם קבצי תמונות ממוזערות משלך לריפו ולשנות את השדה thumb מנתיב data URL לנתיב רגיל של קובץ בריפו.
